# rikksyboi

Personal portfolio website for Rikksy — a full-stack web developer.

## About

A clean, responsive portfolio built with vanilla HTML, CSS, and JavaScript. Features include:

- **Dark / Light mode** toggle
- **Typed text** animation on the hero section
- **Animated skill bars** and counter stats
- **Project showcase** with links
- **Contact form** with client-side validation
- **Scroll reveal** animations

## Tech Stack

- HTML5
- CSS3 (custom properties, flexbox, grid)
- Vanilla JavaScript (no frameworks)

## Getting Started

Open `index.html` in your browser — no build step required.
